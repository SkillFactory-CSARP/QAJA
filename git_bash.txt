Добавил полезные комадны git bash

git clone - копирование репозитория из GitHub
git commit - закрепление сделанных измнений, с открытием редактора
git commit -m "комментарий" тоже самое, только без открытя редактора
git add . - подготавливает файл для дальнейших изменений
git diff - информация о проделанных изменения в файле
git mv - перемещение (переименование файла)
git delete - удаление файла
git log - история изменений файлов
touch имя_файла — Создать файл
touch app/{имя_файла_1, имя_файла_2} — Создать сразу несколько файлов, никаких лишних пробелов быть не должно
git branch - выводит список ветвей
git branch имя_ветви - создает новую ветвь
git git checkout имя_ветви - переключается на указанную ветвь
git merge имя_ветки - проводит слияние
git mergetool имя_файла - разрешат конфликт
